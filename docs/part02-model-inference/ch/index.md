# Part II 模型与推理层

本部分不再强制套用统一三层章节模板，而是按模型与推理层的实际决策链展开：先回答“选什么模型”，再回答“如何本地服务化”，然后进入“如何优化推理成本和延迟”“如何让模型输出可被系统消费”“如何定制能力与接入企业知识”。各章保留章头导读、关键议题、必要图表和本章小结；接口契约、实现路径、发布准入或接入条件只在章节对象确实需要时出现。

## 本部分章节
