# Part IX Frontend, Interaction, and Multimodality

Part IX explains the user-facing side of enterprise Agent systems. The frontend is no longer a chat window that renders text after a model call. It is the surface where task state, tool execution, evidence, approval, artifacts, file parsing, voice interaction, and user feedback become visible.

## Goals of this part

Readers should learn how dialogue UI, streaming events, Generative UI, artifact workspaces, multimodal input, and voice Agents connect to the backend runtime. The emphasis is on controlled interaction: visible progress, recoverable failure, safe rendering, evidence retention, and audit-friendly user actions.

## Chapters in this part

Chapter 47 covers dialogue UI and streaming output. Chapter 48 covers Generative UI and rich interaction. Chapter 49 covers multimodal input and voice Agent design.

## Reading path

Read Chapter 47 before building any Agent frontend protocol. Read Chapter 48 when the product needs editable cards, charts, reports, or generated workspaces. Read Chapter 49 when files, images, audio, or real-time voice enter the Agent task chain.
